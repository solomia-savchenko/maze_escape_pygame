# Maze Escape

Maze Escape is a Python mini-game made with the Pygame library.

The player moves through a maze, finds a key, and opens the door to escape.
The game includes sound effects, background music, animations, and menu screens.

## Controls
- W – Move up
- A – Move left
- S – Move down
- D – Move right

## Assets
All sounds and images are stored in the `assets` folder.
